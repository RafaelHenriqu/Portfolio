# 🌐 Portfólio

---

## 📌 Sobre o projeto

Este é o meu portfólio pessoal, desenvolvido com foco em apresentar minhas habilidades como desenvolvedor de forma visual, interativa e organizada.

A proposta deste projeto foi criar uma experiência mais dinâmica para o usuário, explorando design moderno e pequenas interações visuais.

---

## ✨ Funcionalidades

* Interface moderna e responsiva
* Animações e efeitos visuais
* Elementos interativos escondidos (easter eggs)
* Navegação dinâmica e estilizada

---

## 🧠 Tecnologias utilizadas

* HTML5 → estrutura do projeto
* SASS → organização e estilização avançada do CSS
* JavaScript → interatividade e efeitos dinâmicos

---

## 🎯 Objetivo

O principal objetivo deste projeto foi:

* Criar um portfólio visualmente atrativo
* Praticar desenvolvimento front-end moderno
* Explorar animações e interações com JavaScript
* Melhorar organização de código com SASS
